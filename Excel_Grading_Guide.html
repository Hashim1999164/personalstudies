üî¢ Part 9: Using SUM Function (Easier Way to Add)

üí° There's an EASIER way to add numbers! Use SUM function!

Instead of typing =B2+C2, you can use SUM:

  1. Click on cell D2
  2. Type: =SUM(B2:C2)
  3. Press Enter

What does this mean?

  • SUM = "add these numbers"
  • B2:C2 = "from cell B2 to cell C2" (all cells in between)
  • So it adds B2 + C2 automatically!
‚ö†Ô∏è B2:C2 means "from B2 to C2" - the colon (:) means "through" or "to"

üìà Part 10: Using AVERAGE Function (Easier Way)

Instead of typing =D2/2, you can use AVERAGE:

  1. Click on cell E2
  2. Type: =AVERAGE(B2:C2)
  3. Press Enter

What does this mean?

  • AVERAGE = "find the average of these numbers"
  • B2:C2 = "from cell B2 to cell C2"
  • So it calculates (B2 + C2) √∑ 2 automatically!
‚úÖ BONUS TIP: If you have 3 tests instead of 2, just change C2 to D2: =AVERAGE(B2:D2) - Excel will automatically divide by 3!

üìã Part 11: Copying Formulas (Time Saver!)

üéØ Instead of typing the same formula again and again, you can COPY it!

Method 1: Using Fill Handle (Easiest!)

  1. Click on cell D2 (the one with the formula =B2+C2)
  2. Look at the bottom-right corner of the cell - you'll see a small square
  3. Move your mouse over that small square - the arrow will change to a + sign
  4. Click and HOLD the left mouse button
  5. Drag DOWN to cell D4 (while holding the button)
  6. Let go of the button
  7. Excel will automatically copy the formula and adjust it!

D3 will now have =B3+C3

D4 will now have =B4+C4

Magic! Excel is smart! ‚ú®

Method 2: Copy and Paste

  1. Click on cell D2
  2. Press Ctrl + C (hold Ctrl key, then press C)
  3. Click on cell D3
  4. Press Ctrl + V (hold Ctrl key, then press V)
  5. The formula is copied!

üéì Part 12: Creating a Letter Grade System

Step 1: Add "Grade" Column

  1. Click on cell F1
  2. Type: "Grade"
  3. Press Enter

Step 2: Create Grading Rules

We'll use this system:

  • 90-100 = A (Excellent)
  • 80-89 = B (Good)
  • 70-79 = C (Average)
  • 60-69 = D (Below Average)
  • Below 60 = F (Fail)

Step 3: Use IF Formula for Ali

Click on cell F2, then type this LONG formula:

=IF(E2>=90,"A",IF(E2>=80,"B",IF(E2>=70,"C",IF(E2>=60,"D","F"))))

What does this mean?

  • IF = "check if this is true"
  • E2>=90 = "is the average 90 or more?"
  • If YES ‚Üí give "A"
  • If NO ‚Üí check next condition (80 or more?)
  • And so on...

Press Enter

Ali's average is 87.5, so you should see "B"

‚ö†Ô∏è IMPORTANT: Make sure you type the formula EXACTLY as shown, including all the commas and quotation marks!

Step 4: Copy to Other Students

Use the fill handle method from Part 11:

  1. Click on cell F2
  2. Drag the small square down to F4
  3. Let go
  4. Grades will appear for all students!

‚ùå Part 14: Common Mistakes (How to Avoid Them)

Mistake 1: Forgetting the Equals Sign

WRONG: B2+C2 (Excel thinks this is text, not a formula)

RIGHT: =B2+C2 (The = tells Excel it's a formula)

Mistake 2: Clicking Wrong Cell

Problem: You type in the wrong cell

Solution: Always check which cell is selected (it has a black border)

Mistake 3: Typing Numbers with Words

WRONG: "85 points" (Excel can't do math with this)

RIGHT: 85 (just the number)

Mistake 4: Not Saving

Problem: You work for hours, then close Excel without saving

Solution: Save every 5-10 minutes! Press Ctrl+S (hold Ctrl, press S)
