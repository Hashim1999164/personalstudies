Q1
Explain the history and generations of computers in detail. Discuss the key features of each generation.
Marks: 10
‚úÖ Model Answer:
Mechanical Era (1623-1900):
- Abacus (3000 BC): Wooden beads for addition, subtraction, division, multiplication
- Napier's Bone (17th Century): Cupboard multiplication calculator
- Pascaline (17th Century): First mechanical adding machine with wheels

First Generation (1937-1953):
- Based on vacuum tubes (glass tubes controlling electronic signals)
- High power consumption, limited performance, very expensive
- Used assembly language, translated to machine language
- Punch cards and paper tape for input
- Magnetic tapes and drums for secondary memory
- Mainly used for scientific computation

Second Generation (1954-1962):
- Bell Labs invented transistor (transfers electronic signals)
- Lower cost than first generation
- Magnetic tapes and magnetic disks as secondary memory
- Languages: FORTRAN, COBOL
- Separate input-output processors operating in parallel with CPU

Third Generation (1963-1972):
- Jack Kilby developed Integrated Circuits (IC) - several components on silicon chip
- IBM introduced 360 system
- Smaller size, faster processor, better performance, lower cost
- RAM and ROM replaced magnetic core memories
- Operating Systems and Multi-programming introduced
- Cache and virtual memories introduced

Fourth Generation (1972-1984):
- Microprocessors introduced as CPU (tens of thousands of transistors on chip)
- Semiconductor memory chips as main memory
- Secondary memory: floppy and magnetic tapes
- LANs and WANs introduced
- Languages: C, Graphical user interface
- Apple and IBM computers

Fifth Generation (1984-1990):
- Computers based on Artificial Intelligence/quantum computing
- Parallel processing and multi-processors

Sixth Generation (1990-Present):
- Intel Pentium (55 million transistors on single chip)
- Memory chips up to 1GB, hard drives up to 180GB
- Languages: JAVA
- Windows 95, 98, Linux developed
- Inventions: WWW, HTML, HTTP, DVD, iPod, YouTube
Q2
Discuss the Central Processing Unit (CPU) in detail. Explain its components and functions.
Marks: 10
‚úÖ Model Answer:
Introduction:
The Central Processing Unit (CPU) is the "brain" of the computer. It contains electronic circuits that cause the computer to follow instructions from ROM or from programs in RAM. By following these instructions, information is processed.

Three Main Components:

1. Arithmetic Logic Unit (ALU):
- Where the "intelligence" of the computer is located
- Can add and compare numbers
- To multiply 2 √ó 4, computer adds 2 + 2 + 2 + 2
- Makes decisions by determining if a number is greater, less, or equal to another number
- Processing completed in nanoseconds (billionth of a second)

2. Memory:
Two types contained on a chip:
- RAM (Random Access Memory): Temporary memory that displays information you are working on. Remembers what you see on screen. Can be upgraded.
- ROM (Read Only Memory): Permanent memory installed by manufacturer. Cannot be altered. Determines basic functions like startup, shutdown, placing characters on screen.

3. Control Unit:
- Directs information to proper places in computer
- Sends information to ALU for calculations
- Sends information to memory for storage
- Sends information to printer for printing
- Sends information to screen for display
- Acts like a traffic controller

Function:
The CPU processes all instructions and performs calculations, making it the most important component of the computer system.

Q5
Explain computer storage devices and data measurement units in detail.
Marks: 10
‚úÖ Model Answer:
Storage Devices:
A storage device is a hardware component that keeps data, information and program instructions permanently. There are many types of storage devices available in a range of capacities. Some are portable and some are not.

Types of Storage:

1. Primary/Internal Storage:
- Internal storage device
- Not portable
- Not usually visible unless system unit is open
- Very fast access
- Examples: Hard drive inside computer

2. Secondary Storage:
- External storage devices
- Portable
- Can be seen and removed
- Keeps data when computer is turned off
- Examples: USB flash drives, CD, DVD, external hard drives

3. USB and FireWire Ports:
- Provide connection technology for attaching peripheral devices
- Enable data exchange

Data Storage Units:
Computer stores data in units called bits and bytes. Computer chips have one of two states: off or on. Therefore, a system was developed using only two numbers: 0 (off) and 1 (on). Each switch is called a bit. Bits are grouped together in sets of eight. Each set of eight bits is called a byte. Setting different combinations of those eight "on and off" combinations can stand for letters, numbers, spaces, and symbols. For practical purposes, think of a byte as one character.

Measurement Units:
- 8 bits = 1 byte (represents one character)
- 1024 bytes = 1 Kilobyte (KB) (approximately 1 page of text)
- 1024 Kilobytes = 1 Megabyte (MB) (approximately 1 song)
- 1024 Megabytes = 1 Gigabyte (GB) (approximately 1 movie)
- 1024 Gigabytes = 1 Terabyte (TB) (approximately 1000 movies)

Examples:
- A text document might be 50 KB
- A song might be 5 MB
- A movie might be 1 GB
- A hard drive might be 500 GB or 1 TB

Scenario 2
A student is working on a document in Microsoft Word. When they type, the text appears on the screen. However, when they turn off the computer without saving, all their work is lost.
Questions:
a) Which type of memory was storing the document while the student was typing? (1 mark)
b) Why was the work lost when the computer was turned off? (2 marks)
c) What should the student have done to prevent data loss? (2 marks)
Total Marks: 5
‚úÖ Model Answer:
a) Type of Memory:
RAM (Random Access Memory)

b) Why Work Was Lost:
- RAM is temporary memory that stores data only while the computer is on
- RAM remembers what you see on screen while working
- When computer turns off, all data in RAM is lost because it is volatile memory
- The document was not saved to permanent storage (hard drive)

c) How to Prevent Data Loss:
1. Save the document frequently (every few minutes) using Ctrl+S or File ‚Üí Save
2. Save to hard drive or secondary storage (USB, cloud storage)
3. Enable auto-save feature if available
4. Always save before closing the program or turning off computer
Scenario 3
A research laboratory needs a computer system to measure continuous temperature changes in an experiment. The temperature readings need to be represented as smooth, continuous values rather than discrete steps.
Questions:
a) What type of computer would be most suitable? (2 marks)
b) Explain the difference between this type and digital computers. (3 marks)
Total Marks: 5
‚úÖ Model Answer:
a) Suitable Computer Type:
Analog Computer (or Hybrid Computer)

b) Difference from Digital Computers:
Analog Computers:
- Use continuous analog signals
- Represented by varying voltages
- Smooth, continuous values (like a wave)
- Used in scientific research centers
- Example: Thermometer, speedometer

Digital Computers:
- Store data in terms of digits
- Process in discrete steps from one state to another
- Use binary numbers (0 and 1)
- Letters and words represented in digital formats
- Example: Personal computers, laptops

Key Difference: Analog uses continuous signals (smooth), while digital uses discrete steps (on/off, 0/1).
Scenario 4
A large bank needs a computer system to handle millions of customer transactions daily. The system must be reliable, secure, and able to process data for thousands of users simultaneously. It should cost between $100,000 to $1 million.
Questions:
a) What type of computer system would you recommend? (2 marks)
b) Explain three reasons for your recommendation. (6 marks)
c) What is a client/server network in this context? (2 marks)
Total Marks: 10
‚úÖ Model Answer:
a) Recommended System:
Mainframe Computer System with Client/Server Network

b) Three Reasons:
1. Capacity and Performance: Mainframes are designed to handle massive amounts of data and transactions. They can process data for thousands of users simultaneously, which is essential for a large bank.

2. Reliability and Security: Mainframes are highly reliable systems with built-in redundancy and security features. Banks require systems that rarely fail and protect sensitive financial data.

3. Cost Range: Mainframes cost between $5,000 to $5 million, which fits the bank's budget of $100,000 to $1 million. They provide excellent value for large-scale operations.

4. Scalability: Mainframes can be configured in various sizes (small to medium to large) depending on needs, and can be upgraded as the bank grows.

c) Client/Server Network:
- A network where a central computer (server) holds collection of data and programs
- The server supplies services to PCs, workstations, and other devices (called clients)
- Clients are linked by wired or wireless network
- The entire network is called a client/server network
- In the bank's case: Mainframe acts as server, bank employees' computers and ATMs act as clients
